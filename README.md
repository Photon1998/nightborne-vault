
# 奈特的秘密金庫 · 密碼破解儀

楓之谷 M「打開奈特的金庫！」活動的密碼推理工具（Mastermind solver）。

密碼為 **1~9 之中互不重複的 3 位數**（共 504 種）。每次輸入後系統回饋：

- `●` 數字與位置皆完全一致
- `▲` 僅數字一致（位置不正確）
- `X` 完全不在密碼中

工具採用 **minimax** 策略推薦下一手。實測全部 504 種密碼：**最壞 7 手、平均 4.78 手**開鎖，穩過 15 手上限。

## 網頁版

開啟 [`index.html`](index.html) 即可使用。刻度盤會顯示要在遊戲裡輸入的密碼，你只要點回報 `●` 與 `▲` 的數量，它就轉出下一組最佳密碼。

功能：即時候選數統計、排除不含的數字、撤回上一步、深／淺主題、手機可用。

## 終端機版

```bash
python3 solver.py
```

腳本直接叫你猜某組數字，你只回報 `●` `▲`：

| 回饋 | 意思 |
|------|------|
| `1 1` | ● 1 個、▲ 1 個 |
| `0 2` | ● 0 個、▲ 2 個 |
| `X` | 全部不中 |
| `no 258` | 系統提示時排除數字 2、5、8 |

## 原理

- 產生所有 504 種合法密碼作為候選集
- 每次回饋後，只保留「若它是答案就會給出相同回饋」的候選
- 以 minimax 選出「最壞情況下能砍掉最多候選」的下一手
